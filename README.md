# KP4
## Краткое описание приложения
Приложение представляет собой веб-калькулятор с базовым функционалом для выполнения арифметических операций: сложение, вычитание, умножение и деление. Приложение разработано с использованием React и демонстрирует работу с компонентами, состоянием и обработкой событий.

## Основные возможности
- Ввод чисел с помощью кнопок
- Выполнение четырех базовых арифметических операций
- Отображение текущего выражения и результата
- Очистка дисплея
- Работа с десятичными числами
- Обработка ошибок (деление на ноль)
